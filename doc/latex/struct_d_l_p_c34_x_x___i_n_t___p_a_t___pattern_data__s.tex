The 1-\/D pixel data array.

For 1-\/bit patterns, each byte should 0 or 1. For 8-\/bit patterns, each byte can be 0-\/255. For horizontal patterns, the number of bytes should be equal to the DMD\+\_\+\+HEIGHT For vertical patterns, the number of bytes should be equal to the DMD\+\_\+\+WIDTH If the number of bytes is less than the expected value, the missing pixel data will be filled with zeros.
