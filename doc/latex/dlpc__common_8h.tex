Initializes the read/write buffers and callbacks for the command APIs

A write command API packs command data to the Write\+Buffer. A read command API packs the read request data to the Write\+Buffer and unpacks the read response data from the Read\+Buffer. To avoid dynamic memory allocation or an arbitrarily large static buffer allocation, the command APIs use the buffers provided the caller.

The command APIs do not directly transmit data to the controller or receive data from the controller. Instead, they use the Write\+Command\+Callback and Read\+Command\+Callback functions provided by the caller to send data to and receive data from the controller.
