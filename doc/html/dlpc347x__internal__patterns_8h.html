Generates the pattern data block from the given inputs. In order to avoid dynamic memory allocation, this function uses a callback to transfer data to the caller as it generates pattern data. This function calls the WritePatternDataCallback function several times with one or more bytes of pattern data transfered to the caller at a time. The function is synchronous and waits for the callback to finish execution before continuing generation of the rest of the data.

Parameters
[in]DMDThe DMD for which pattern data is being generated
[in]PatternSetCountNumber of pattern sets
[in]PatternSetArrayAn array of DLPC34XX_INT_PAT_PatternSet_s
[in]PatternOrderTableCountNumber of rows in the pattern order table
[in]PatternOrderTableAn array of DLPC34XX_INT_PAT_PatternOrderTableEntry_s
[in]WritePatternDataCallbackThe callback used to transfer data to the caller
[in]EastWestFlipWhether to E/W flip pattern data
[in]LongAxisFlipWhether to flip pattern data along the long axis
Returns
DLPC_SUCCESS if successful ERR_UNSUPPORTED_DMD if the DMD is not supported
